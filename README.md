# Proyecto: Inferencia Estadística sobre Hábitos Saludables

## Descripción
Este proyecto analiza los hábitos de sueño, alimentación y actividad física en estudiantes universitarios utilizando técnicas de inferencia estadística.

## Objetivo
Aplicar el método científico para identificar patrones y generar recomendaciones de bienestar estudiantil.

## Contenido
- Simulación de datos
- Análisis estadístico
- Intervalos de confianza
- Pruebas de hipótesis
- Visualizaciones

## Tecnologías
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

## Resultados
Se identificó relación significativa entre hábitos saludables y bienestar general.
